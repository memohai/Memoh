# Memoh 文档

Memoh(/ˈmemoʊ/) 是一个多成员、结构化长记忆、容器化的 AI Agent 系统。创建你自己的 AI Bot，通过 Telegram、飞书、Web 或 CLI 与它们聊天。每个 Bot 运行在独立的容器中，拥有自己的记忆系统——能够编辑文件、执行命令和访问网络。
