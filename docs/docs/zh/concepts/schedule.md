# 定时任务

**定时任务（Schedule）** 是基于 cron 的自动化规则，在指定时间运行 Bot 命令。

## 核心字段

- `pattern`：cron 表达式
- `command`：要执行的命令或任务
- `enabled`：定时任务是否启用
- `max_calls`：可选的执行次数上限
- `current_calls`：当前已执行次数

## 为什么重要

定时任务使 Bot 变得主动，而不仅仅是被动响应。
你可以自动化重复性操作，如摘要生成、提醒、检查和维护任务。
